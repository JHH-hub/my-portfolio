项目概述 Overview

亚森科技（Ascen Technology）是一家专注于载带生产与电子封装材料的制造型企业。 本项目对其品牌形象进行了系统化的重构。过往的形象偏向传统工程属性，本次设计通过“结构化线条 + 理性构图 + 科技蓝”的视觉体系，提升了企业的数字化专业形象。

数字化体验 Web Design

我们构建了全新的企业官网，以模块化排版展示产品规格、工艺流程与应用场景。通过清晰的信息架构（Information Architecture），优化了 B 端客户的浏览体验。

官网 UI 设计与响应式布局

品牌宣传片 Promotional Video

独立完成了宣传片的脚本撰写、建模、动效与剪辑，将技术参数与产品卖点可视化。

物料延展 Applications

设计了名片、产品手册、包装贴纸等全套物料，建立了高度统一的品牌视觉识别系统 (VI)。
